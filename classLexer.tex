\doxysubsection{Detailed Description}
The \doxylink{classLexer}{Lexer} class is responsible for reading input, applying lexer rules, and generating tokens from the input. 

The \doxylink{classLexer}{Lexer} class provides methods to load configuration rules, tokenize input strings, and generate production rules from files.
