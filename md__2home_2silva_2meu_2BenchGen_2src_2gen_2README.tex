\hypertarget{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md3}{}\doxysection{\texorpdfstring{Running}{Running}}\label{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md3}
Run the binary output with a input and output file as parameter.

Calls to the Program \doxylink{classGenerator}{Generator} must be done in the following format\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./main\ <Number\ of\ iterations>\ <Production\ Rules\ File>\ <Seed\ String\ File>\ <Output\ Folder>\ <Variable\ Type>}

\end{DoxyCode}


The available variables types are in the {\ttfamily \doxylink{varTypes_8h}{shared/var\+Types.\+h}} file.\hypertarget{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md4}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md4}
We provide examples for inputs in the {\ttfamily examples folder}. An example of execution would be\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./main\ 3\ production\_rule\_8.txt\ seed\_string.txt\ output\ array}

\end{DoxyCode}
\hypertarget{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md5}{}\doxysubsection{\texorpdfstring{Production Rules File}{Production Rules File}}\label{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md5}
The production rules file should have production rules of the format\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<LHS>\ =\ <Production>;}

\end{DoxyCode}


You may repeat these as many times as you wish for each production rule.

Example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{A\ =\ insert\ A\ remove;}
\DoxyCodeLine{B\ =\ A\ B;}

\end{DoxyCode}
\hypertarget{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md6}{}\doxysubsection{\texorpdfstring{Seed String File}{Seed String File}}\label{md__2home_2silva_2meu_2BenchGen_2src_2gen_2README_autotoc_md6}
The Seed String File should contain only {\bfseries{Valid Tokens}}. Other than that, these tokens can be written as the user wishes.

Example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{IF(LOOP(CALL(B)),A)}

\end{DoxyCode}
